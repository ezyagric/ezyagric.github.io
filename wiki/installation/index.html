Installation

To setup this app you will need the main app as well as three other modules:-

  • ezyagric_db for database management. github
  • ezyagric_core for base classes & helpers. github
  • ezyagric_ui for loaders and UI related re-usables. github

Note if you can't access the libraries, request DevOps team to provide you access to their repos

Building the App

First, clone the repo:

git clone https://github.com/ezyagric/ezyagric_farmer_app_2020.git

Note you can clone with libraries and skip the next step entirely

Setting up the custom libraries

Next, you will need to have the module libraries. This can be done at clone time or later using the automated method, however if it fails you can try the manual method.

During Repo Clone

The modules can be fetched by running the command

git clone --recurse-submodules https://github.com/ezyagric/ezyagric_farmer_app_2020

Automatic method

If you already cloned the project and forgot --recurse-submodules, the modules can be fetched by running the command

git submodule update --init

Manual method

If all the above two methods have failed then you'll have to manually setup the modules

ezyagic_core

cd libraries/ezyagric_core
git checkout v1.0.0

ezyagic_db

cd libraries/ezyagric_db
git checkout master

ezyagic_ui

cd libraries/ezyagric_ui
git checkout v1.0.0

Note: To update already existing modules use the command git submodule update --init --recursive And to skip one use git -c submodule."libraries/ezyagric_doc".update=none submodule update --init --recursive If your having trouble with the ezyagric_doc module, simply comment it out of settings.gradle

include ':libraries:ezyagric_db'
include ':libraries:ezyagric_core'
include ':libraries:ezyagric_ui'
//include ':libraries:ezyagric_doc'
include ':app'
rootProject.name='EzyAgricFarmer'
